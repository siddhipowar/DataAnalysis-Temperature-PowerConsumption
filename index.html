Temperature Testing on Gordon


Conclusion of testing:


  • At same configuration, temperature for short integration time < temperature at long integration times
  • Temperature increases with increasing frame rates
  • Observed temperatures order for config types is: speed<balanced<high accuracy
  • Power consumption is proportional to temperature rise
  • Config type 'speed' tries to capture frames up to approximately the maximum limit set to frame rate. While other config types are observed to capture half of FPS limit set (for example: in balanced, observed FPS= 20.2 AND SET= 40)
